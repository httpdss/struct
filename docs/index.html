Struct

Your versatile project structure generator

Introduction

Struct is a powerful tool to automate the creation of project structures based on YAML configurations.

Features

  • Define project structure using YAML
  • Support for template variables
  • Integration with OpenAI for content generation
  • Flexible file handling strategies
  • Modular YAML inclusion
  • Dry-run mode
